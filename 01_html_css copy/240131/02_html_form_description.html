입력양식 태그

fieldset

입력 양식을 그룹으로 묶어주는 태그(disabled)

label

입력 양식의 설명을 만들어주는 태그(for: 입력 양식의 id와 연결)

form 태그

form 태그는 입력 양식 폼을 만들어주는 태그

form 태그의 속성

  • action: 입력 데이터를 전송할 서버의 주소 지정(상대 경로, 절대 경로)
  • method: 입력 데이터를 전송할 방식을 지정 (GET, POST)

input 태그

input 태그의 속성

  • type

    입력 양식의 종류를 지정(text, password, number, range, color, email, tel, url, search, date, time, week, datetime-local, checkbox, radio, file, image, submit, reset, button, ...)

  • name

    입력한 데이터를 전송하기 위해 지정(값에 이름을 붙인다.[name=value])

  • value

    입력 양식의 값을 의미하고, 지정하면 기본값으로 설정된다.

  • placeholder, required, readonly, disabled, ...

    입력 양식의 힌트, 필수 여부, 읽기 전용 설정, 사용불가 설정 등을 할 수 있다.

select

여러 개의 선택지를 가진 입력 양식을 생성

option

select, datalist 태그의 선택지를 만들어주는 태그(selected, name, value, disabled)

textarea

여러 줄의 텍스트를 입력할 수 있는 입력 양식(cols, rows 속성으로 크기 설정)

button

버튼을 생성(type, name, disabled), 빈 태그가 아니다.
