목록(list) 태그

(ol, li), (ul, li), (dl, dt, dd)는 목록을 만드는데 사용됩니다.

(ol, li)

번호가 붙은 목록을 만드는데 사용한다.

  1. ol(ordered list) - 순서가 있는 목록
  2. li(list item) - 목록 항목

(ul, li)

점으로 나타나는 목록을 만드는데 사용한다.

  • ul(unordered list) - 순서가 없는 목록
  • li(list item) - 목록 항목

(dl, dt, dd)

용어와 해당 설명을 정의하는 데 사용된다.

dl(description list)
설명 목록을 의미합니다.
dt(description term)
설명 용어를 의미합니다.
dd(description definition)
설명에 대한 정의를 의미합니다.

2단계 리스트 구성

  • 메뉴 1
    • 하위메뉴 1
    • 하위메뉴 2
    • 하위메뉴 3
  • 메뉴 2
    • 하위메뉴 1
    • 하위메뉴 2
    • 하위메뉴 3
