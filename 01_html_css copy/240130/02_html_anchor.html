앵커 태그

정의

앵커(a) 태그는 링크를 만들어주는 태그입니다.

속성

  • href: 링크할 주소를 지정
  • target: 링크를 열 위치를 지정

href 속성

  • 링크할 주소를 지정한다.
  • 상대경로와 절대경로 모두 사용 가능하다.
  • 다른 페이지로 이동할 수 있게 한다.
  • 페이지 내부에서 특정 위치로 이동할 수도 있다.
  • 이메일과 전화번호도 연결 가능하다.

target 속성

  • 링크를 열 위치를 지정한다.
  • 기본값은 _self이다.
  • _self는 현재 창에서 링크를 연다.
  • _blank는 새 창에서 링크를 연다.

target="_blank"의 보안 문제

  • target="_blank"는 보안취약점(정보 노출)이 존재할 수 있다.
  • 브라우저의 팝업 차단기에 의해 차단될 우려가 있다.
  • target="_blank"에 rel="noopener noreferrer"을 추가하면, 외부 페이지에서 현재 페이지에 대한 악의적인 조작을 방지하고, 사용자의 개인정보를 보호할 수 있어서 보다 안전하게 처리된다.
noopener
새 창, 새 탭을 열 때, 부모 창에 대한 참조를 막는다.
새로운 창이 부모 창에 대한 제어권을 가질 수 없게 한다. (보안문제 제거)
스크립트가 새로 열린 창의 내용을 조작할 수 없게 한다.
noreferrer
HTTP 리퍼러 헤더를 제거한다.
리퍼러 헤더는 현재 페이지에서 새로 열린 페이지로 전달되는 정보이다.
noreferrer를 쓰면, 외부 사이트가 어디에서 링크를 클릭했는지 알 수 없게 한다.

  • HTML 스타일
